Tentang Kami


MBC (Multimedia, Big Data & Cybersecurity) Laboratory merupakan salah satu laboratorium yang berada di Telkom University di bawah naungan KK NCM. Laboratorium ini terbentuk pada tanggal 4 Oktober 2019


Laboratorium ini memiliki dari 4 divisi, yaitu Geographic Information System (GIS), GameTech, Big Data dan Cyber Security.

Geographic Information System (GIS)


Divisi GIS fokus pada pengembangan sistem informasi geografis dan pemetaan digital. Kegiatan utama termasuk analisis spasial, pengembangan aplikasi berbasis peta, dan integrasi data geografis dengan teknologi terkini.

GameTech


Divisi GameTech berfokus pada pengembangan game dan teknologi interaktif. Kegiatan termasuk pembuatan game edukasi, pengembangan engine game, dan eksplorasi teknologi realitas virtual/augmented.

Big Data


Divisi Big Data mengkhususkan diri dalam analisis dan pengolahan data skala besar. Kegiatan meliputi data mining, machine learning, visualisasi data, dan pengembangan sistem rekomendasi.

Cyber Security


Divisi Cyber Security berfokus pada keamanan sistem dan jaringan. Kegiatan termasuk penetration testing, pengembangan tools keamanan, analisis malware, dan edukasi keamanan digital.
